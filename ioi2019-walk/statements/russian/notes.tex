\textbf{Пример 1}
Рассмотрим следующий вызов:

\begin{verbatim}
min_distance([0, 3, 5, 7, 10, 12, 14],
             [8, 7, 9, 7, 6, 6, 9],
             [0, 0, 0, 2, 2, 3, 4],
             [1, 2, 6, 3, 6, 4, 6],
             [1, 6, 8, 1, 7, 2, 5],
             1, 5)
\end{verbatim}

Правильный ответ равен $27$.
